# System Design Visualizer

An interactive system architecture designer built with Next.js, React Flow, TypeScript, and Tailwind CSS.

## 🚀 Features

- Add system components
- Connect components with interactive edges
- Drag and move nodes
- Custom node design
- Zoom and pan support

## 🛠️ Technologies

- Next.js
- React
- TypeScript
- React Flow
- Tailwind CSS

## ⚙️ Installation

```bash
git clone https://github.com/<your-username>/system-design-visualizer.git

cd system-design-visualizer

npm install

npm run dev
```

Then open:

```
http://localhost:3000
```
